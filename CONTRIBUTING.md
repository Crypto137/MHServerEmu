# Contributing to MHServerEmu

- Please ask questions before making changes and creating pull requests. You can do it on [Discord](https://discord.gg/hjR8Bj52t3) or by creating an [issue](https://github.com/Crypto137/MHServerEmu/issues).

- If you have performance improvement suggestions, include benchmark results and/or profiler data.

- Do not create pull requests for Docker implementations.
